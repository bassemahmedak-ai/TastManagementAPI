# Task Management Web App

A simple full-stack task management system built using ASP.NET Core Web API.

## Features
- Add tasks
- Delete tasks
- View tasks
- RESTful API
- SQLite database

## Tech Stack
- ASP.NET Core
- Entity Framework Core
- SQLite
- HTML, CSS, JavaScript

## How to Run
1. Open the project in Visual Studio
2. Run the application
3. Open https://localhost:7239
